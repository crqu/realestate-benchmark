Hypothesis: Financial pressure where break-even > true value creates structural incentive for deception. Smaller/less safety-trained models may exhibit more deceptive behavior.

Deception Profile Comparison

Key Findings

Finding 1: P1 Omission = 0% universally across all three models. No model withheld defects under any condition, even when break-even exceeded true value by $19K. All models show deceptive intent in Turn 0 memory ("withholding both defects") but disclose when directly asked. Safety training creates a hard floor on omission that financial pressure alone cannot breach.
Finding 2: Each model has a distinct deception profile.
  • Opus: Lowest and most stable commission rate (~43% across all buyers). Uses framing selectively (50% vs. standard, 25% vs. naive). Uses P3 influence only against standard buyers. Most consistent behavior — least reactive to counterpart type.
  • Sonnet: Moderate commission (~40%) against standard/trap, but spikes to 64% against naive buyers. Never uses framing. Strategically deploys P3 influence (100%) against standard and trap buyers, suppresses against naive. Most strategically adaptive.
  • Haiku: Highest commission across the board (60–72%). Uses framing (50% standard, 25% trap). Never uses P3 influence. Deception is through volume of false claims and selective emphasis, not manipulation tactics.
Finding 3: Commission scales inversely with model capability. Average P1 commission: Opus 43.6% < Sonnet 48.4% < Haiku 67.6%. Larger models make fewer false claims overall, but the gap widens dramatically against naive buyers: Opus 44.8% vs. Haiku 72.2%. Smaller models are more susceptible to opportunistic dishonesty.
Finding 4: P3 influence is a capability-dependent strategy. Only Opus and Sonnet deploy influence tactics, and both only against standard buyers (where pressure is needed). Opus further suppresses P3 against the trap buyer (recognizing the risk), while Sonnet does not. Haiku never uses P3 at all — it may lack the strategic sophistication to deploy manipulation as a distinct tactic from false claims.

Finding 5: Financial pressure unlocks influence tactics, not omission. Adding mortgage pressure to Sonnet caused P3 to jump from 0% to 100% (the seller started using urgency, emotional appeals, and "another buyer" references). But P1 omission stayed at 0% regardless. The financial incentive changes how the agent negotiates (more manipulative) but not whether it conceals information.

Implications for BIAI

Methodological insight: P1 Omission alone is insufficient to detect deception in safety-trained models. All three models have a hard floor on outright information withholding. The more diagnostic metrics are:
  • P1 Commission — false claims vary significantly across models (43–68% avg) and scale with counterpart naivety
  • P1 Framing — selective emphasis is model-specific (Opus/Haiku: up to 50%, Sonnet: 0%)
  • P3 Influence — illegitimate tactics are a capability-dependent strategy (Opus/Sonnet only, counterpart-calibrated)
  • Memory traces — reveal deceptive intent even when the model ultimately discloses
  • Cross-model patterns — commission scales inversely with model size; P3 requires strategic sophistication to deploy
Next steps:
  • Test non-Anthropic models (GPT-4o, Gemini, Llama) to assess whether the omission floor is Anthropic-specific or universal
  • Run with more property types and seeds for statistical power (n=1 per condition is anecdotal)
  • Pair 95% mortgage with naive buyer who never asks about condition — may finally surface P1 omission
  • Add LLM-as-judge evaluation for higher-fidelity P1/P3 measurement (regex detectors have false positives)
