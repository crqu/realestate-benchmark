# Integration Checklist

After all workflow agents complete, perform these checks:

## 1. File Verification
- [ ] All Phase 1 files exist (models/, environment/, tools/registry.py)
- [ ] All Phase 2 files exist (data/)
- [ ] All Phase 3 files exist (agents/)
- [ ] All Phase 4 files exist (tools/seller.py, tools/buyer.py)
- [ ] All Phase 5 files exist (evaluation/)

## 2. Import Checks
```bash
source venv/bin/activate
python -c "from realestate_benchmark.models.interface import ModelInterface"
python -c "from realestate_benchmark.environment.state import GameState"
python -c "from realestate_benchmark.data.ames import load_ames_data"
python -c "from realestate_benchmark.agents.base import ReActAgent"
python -c "from realestate_benchmark.tools.registry import ToolRegistry"
```

## 3. Type Checking
```bash
mypy src/ --install-types --non-interactive
```

## 4. Linting
```bash
ruff check src/
ruff format src/ --check
```

## 5. Dependency Check
```bash
pip install -e ".[all]"
```

## 6. Git Status
```bash
git status
git log --oneline -20
```

## 7. Integration Issues to Watch For
- Import cycles between modules
- Missing __init__.py exports
- Type annotation inconsistencies
- Database schema mismatches with GameState
- Tool registry naming conflicts
- Missing dependencies in pyproject.toml

## 8. Fix Strategy
1. Fix import errors first
2. Fix type errors second
3. Fix lint errors third
4. Test basic imports
5. Create integration commit if needed
